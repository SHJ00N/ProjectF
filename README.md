# Dark Knights
This project is a 3D game developed to learn and practice OpenGL and graphics theory. The game is a simple action game where you defeat enemies

<img width="1433" height="1072" alt="readme" src="https://github.com/user-attachments/assets/dbda4839-c8ac-453a-abae-8b64998fa12e" />

### Controls

WASD: movement\
Space: roll\
Mouse Left: attack\
Mouse Wheel: zoom
ESC: exit game
